## 🤝 Contributing

We welcome contributions to PolyGuard.

## 📌 How to Contribute
1. Fork the repository
2. Create a new branch
3. Make changes
4. Commit changes
4. Push to GitHub
5. Open a Pull Request


## 🧪 Guidelines
- Write clean and readable code
- Follow project structure
- Test your changes


## 🐞 Reporting Issues
- Use GitHub Issues
- Clearly describe the problem
- Provide steps to reproduce

## 🔚 Summary

Contributions are encouraged. Follow the process and maintain quality.
